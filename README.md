[Status](http://status.versidyne.com/) - Simple Status Page

Simple status page for services running on a server.

Services:

* Apache
* LigHTTPD

Interpreters:

* PHP5

#####Demonstration
Website: [http://status.versidyne.com/](http://status.versidyne.com/)   

#####Installation
